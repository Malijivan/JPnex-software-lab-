## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/jpnex-software-lab.git
   ```
2. Navigate to the project directory:
   ```bash
   cd jpnex-software-lab
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```
The server will run at `http://localhost:3000`.

### Production Build
To create a production-ready build:
```bash
npm run build
```
And to start the production server:
```bash
npm run start
```

## SEO & Deployment
- The site uses an auto-generated `sitemap.xml` and `robots.txt` in the `/public` directory to ensure perfect Google Search Console indexing.
- Update `/public/sitemap.xml` periodically when adding new public routes.
